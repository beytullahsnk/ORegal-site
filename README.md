# O'RÉGAL - Version HTML/CSS/JS

Version statique du site, prête à être déployée sur n'importe quel hébergeur.

## Fichiers

- `index.html` - Page d'accueil
- `commander.html` - Page de commande
- `styles.css` - Tous les styles
- `hero-bg.png` - Image de fond du hero
- `.htaccess` - Configuration Apache (redirections)

## Déploiement

1. Upload tous les fichiers dans le dossier `httpdocs` (ou `public_html`) de ton hébergement
2. Assure-toi que `hero-bg.png` est bien présent
3. C'est tout ! Le site fonctionne sans Node.js ni build

## Structure

```
html-version/
├── index.html
├── commander.html
├── styles.css

```
